# MemoryGame
- 記憶遊戲 - Discord.js
- 中文機器人 - 記憶遊戲

# 如何使用?
## Step 1:
- 安裝 Node.js
- https://nodejs.org/dist/v20.16.0/node-v20.16.0-x64.msi
- 點擊此連結直接下載 Node.js

## Step 2:
- 解壓縮 "node_modules.tar.gz"

## Step 3:
- 打開 "config.json" 並將裡面的 "YOUR_TOKEN" 和 "YOUR_CLIENTID" 改成你的機器人的 "Token" 和 "Client ID"

## Step 4:
- 開啟 "start.bat"

## 接著到那個機器人在的伺服器, 到一個可以使用指令的頻道
- 輸入指令 **/小遊戲 遊戲: 記憶遊戲**
- 就可以開始玩了!
